1. **homeprofile.png** - Hero section profile image (400x400px recommended)
2. **me.jpg** - About section profile image (500x500px recommended)

## Project Images
Place the following images in `public/assets/projects/`:

1. **happycraft.jpg** - HappyCraft Event LLP project thumbnail (800x600px recommended)
2. **ket-design.jpg** - KET Design & Interior project thumbnail (800x600px recommended)
3. **staffworks.jpg** - StaffWorks project thumbnail (800x600px recommended)
4. **mudd-vision.jpg** - Mudd Vision project thumbnail (800x600px recommended)

## Temporary Placeholders
Until you add your actual images, you can use placeholder images from:
- https://placehold.co/800x600/png
- https://via.placeholder.com/800x600
- Or any other placeholder service

## Image Guidelines
- Use high-quality images (at least 800px wide for projects)
- Optimize images for web (compress using tools like TinyPNG)
- Use JPG for photos, PNG for images with transparency
- Keep file sizes under 500KB when possible
